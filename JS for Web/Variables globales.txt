En javascript, toute variable n'étant pas déclarée dans une fonction est une variable globale : elle est accessible dans tous le script. Si un conflit survient entre le nom d'une variable globale et le nom d'une variable locale, la variable locale prend le dessus tant que l'exécution a lieu dans son scope.

Le contenu des variables globales peuvent être modifiées par l'utilisateur via la console.
